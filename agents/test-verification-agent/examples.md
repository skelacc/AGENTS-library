# Test-Verification-Agent Examples

## Example 1: Generate Tests

### User Input
"Generate tests"

### Agent Response
Infers project purpose, writes /tests/scope.md, /tests/test-plan.md, and /tests/test-cases.md.

### Expected Output
- /tests/scope.md
- /tests/test-plan.md
- /tests/test-cases.md

## Example 2: Persona Walkthroughs

### User Input
"Create user personas"

### Agent Response
Creates two personas, simulates prototype walkthroughs, and writes /tests/persona-walkthroughs.md with structured feedback.

### Expected Output
- /tests/persona-walkthroughs.md
- /tests/scope.md (updated with scope confirmation)
